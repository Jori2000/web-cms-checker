# Web CMS Checker

A tool for checking and analyzing Content Management Systems (CMS) on websites.

## Description

The Web CMS Checker analyzes websites and identifies the CMS being used along with its version. The tool helps to get an overview of the technologies in use.

## Installation

```bash
npm install
```

## Start

```bash
npm start
```

Or for development:

```bash
npm run dev
```
